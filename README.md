🏨 Chef's Hotel Website
📌 Overview

Chef’s Hotel Website is a modern, responsive landing page designed for a luxury hotel.
It highlights rooms, services, and contact details with a clean and elegant user interface.

This project is perfect for showcasing HTML & CSS frontend skills in a real-world hospitality use case.

🌟 Features
✨ Attractive hero section with background image
🛏️ Rooms section (Deluxe & Suite rooms)
🍽️ Services section (Restaurant, Spa, Free Wi-Fi)
📞 Contact section
📱 Responsive design (basic layout)
🎨 Clean and minimal UI
🛠️ Tech Stack
HTML5
CSS3
